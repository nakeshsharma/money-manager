## Overview
Money Manager is a **full-stack web application** designed to help users manage income, expenses, and categories effortlessly. It features secure authentication, real-time transaction tracking, data filtering, and insightful data visualization dashboards‚Äîall powered by Java, Spring Boot, MySQL, React, and JWT for a robust and secure experience.
